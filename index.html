Get the edge in cryptocurrency trading!

trademindx is a system that aims to facilitate daily trading decisions through applied natural-language processing, sentiment analysis and machine learning algorithms.

Built as a micro-services based distributed architecture, its primary function is to analyse large volumes of data and provide clear indicators to the trader in real-time.

The default approach is implemented using a maximum entropy algorithm with feedback and is currently being developed specifically for cryptocurrency applications.

Trading signals at your fingertips

Big Data

The system combines natural-language processing, sentiment analysis and machine learning algorithms to classify large volumes of data including news, exchange prices and even Tweets.

Machine Learning

By design, the system is in a state of continuous automated machine learning, constantly adjusting and updating its internal models and improving accuracy.

Clear Signals

Simple BUY, SELL or HOLD indicators for each cryptocurrency (with a market capitalisation over a $100mm) are provided to you with a clear degree of confidence.

Proven Tech

Built on proven technologies including Apache's OpenNLP, VADER (Valence Aware Dictionary and sEntiment Reasoner), Lucene, Java, MongoDB and Spring Boot.

Frequently Asked Questions


Basics

What is the TMX token?

The TMX token is a utility token based on the Ethereum blockchain.

Is it a security?

The TMX token is NOT a digital currency, security, commodity or any other kind of financial instrument.
The TMX token is a utility token to be used in conjunction with the trademindx platform.

What is TMX used for?

The TMX token grants access to the trademindx platform.
Ownership of tokens carries no rights other than the right to use them as a means to enable usage of and interaction with the trademindx platform, if successfully completed and deployed. The tokens do not represent or confer any ownership right or stake, share or security or equivalent rights, or any right to receive future revenue shares, intellectual property rights or any other form of participation in or relating to the trademindx platform. The tokens are not refundable. We also envision that third-party developers may come up with new and novel use cases for the token.

Is TMX transferable?

TMX is a utility token meant for use on the trademindx platform.
By virtue of being built on top of the Ethereum blockchain, it is transferable.

Which wallets can be used to store TMX?

TMX is an ERC20 token, it can be stored in ERC20 compatible wallets.
While we do not recommend any particular product, some users report storing their TMX using a Ledger Nano S hardware wallet and MyEtherWallet.

Is TMX listed on any exchanges?

The token maybe listed on exchanges in the future however this is not guaranteed at this time.
The purposes of listing TMX on exchange is to provide liquidity and allow the token to be exchanged for other tokens.

What will be the value of TMX tokens in the future?

The TMX token is a utility token for use on the trademindx platform only.
We make absolutely no comment regarding the current or future value of TMX tokens.

Show me the TMX token code!

Yes, the TMX token smart contract code is written in Solidity and the source code can be found on GitHub
